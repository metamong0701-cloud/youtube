# Banana Studio - AI Character Image Generator

Step 2에서 구현한 웹 로직 및 페이지 플로우입니다.

## 구현된 기능

### 1. 싱글 페이지 애플리케이션 구조
- **설정(입력) -> 생성(처리) -> 결과(출력)** 플로우를 한 화면에서 처리

### 2. 상단 (Header)
- 🍌 Banana Studio 로고

### 3. 좌측/중앙 (입력 영역)
- **캐릭터 파일 등록**
  - 파일 업로드 버튼 (이미지 파일만 허용)
  - 이미지 미리보기 화면
  - 기본 캐릭터로 저장 버튼 (LocalStorage 활용)
  
- **프롬프트 입력**
  - 텍스트 입력창
  - Enter 키로 빠른 생성 지원 (Shift+Enter는 줄바꿈)

- **이미지 생성 버튼**
  - 입력 검증 포함

### 4. 우측/하단 (출력 영역)
- **초기 상태**: 안내 메시지
- **로딩 상태**: 스피너 애니메이션 + "AI가 그림을 그리는 중..."
- **결과 화면**: 생성된 이미지 표시 + 다운로드 버튼

### 5. 내부 데이터 흐름
1. 사용자가 이미지(A)와 명령어(B) 입력
2. `generateImageWithGemini()` 함수로 API 호출 준비
3. AI 처리 (현재는 시뮬레이션, 실제 Gemini API 연동 필요)
4. 생성된 이미지(C) 화면에 표시
5. 다운로드 기능 (`character_result.png`)

## 실행 방법
1. `index.html` 파일을 브라우저에서 열기
2. 캐릭터 이미지 업로드
3. 프롬프트 입력
4. 이미지 생성 버튼 클릭
